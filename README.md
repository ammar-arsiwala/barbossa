# Barbossa
## Getting Started

### Building Engine
```bash
$ go build && ./barbossa
```

## Contributing
1. Fork the repo
2. Pull the forked repo
```
git pull git@github.com/<yourname>/barbossa
```
or
```
git pull https://github.com/<yourname>/barbossa
```

3. Create new branch
```
git checkout -b "branch name"
```

4. Push to your repo
```
git push origin master
```

5. Create a PR for the branch
